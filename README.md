#Prerequisite

1) JDK 21
2) Spring Boot v3.3.4
3) MS SQL Server
4) DB name: TESTDB
5) Schema name: dbo (default)
6) Execute the required DDL and DMLs from the db folder
7) Before starting the app, kindly change the database endpoint in application.properties
